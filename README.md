# الحبيب للهندسة والمقاولات - موقع إلكتروني

**Al Habeeb Engineering & Contracting - Official Website**

موقع إلكتروني فخم وحديث لشركة الحبيب للهندسة والمقاولات، المتخصصة في التطوير العقاري والمشاريع الهندسية الكبرى بصعيد مصر.

---

## 📋 نظرة عامة

هذا المشروع عبارة عن موقع ويب متقدم مبني باستخدام أحدث التقنيات، يعكس الفخامة والاحترافية لشركة الحبيب. الموقع يتميز بتصميم سينمائي، حركات سلسة، دعم كامل للغة العربية (RTL)، وتجربة مستخدم استثنائية.

### ✨ المميزات الرئيسية

- **تصميم فخم:** نظام ألوان داكن مع لمسات ذهبية (Navy & Gold)
- **حركات متقدمة:** استخدام Framer Motion لحركات سينمائية احترافية
- **دعم RTL كامل:** تصميم محسّن للغة العربية من البداية
- **استجابة كاملة:** تصميم متجاوب يعمل على جميع الأجهزة
- **أداء عالي:** تحسينات SEO وتحميل سريع
- **TypeScript:** كود آمن وموثوق مع نظام أنواع قوي

---

## 🛠️ التكنولوجيا المستخدمة

### Frontend Stack
- **Next.js 15** - إطار عمل React حديث مع Server-Side Rendering
- **React 18** - مكتبة واجهات المستخدم
- **TypeScript** - لغة برمجة آمنة مع نظام أنواع
- **Tailwind CSS 3** - نظام تصميم فعال وقابل للتخصيص
- **Framer Motion** - مكتبة حركات متقدمة
- **Lucide React** - مجموعة أيقونات عصرية
- **React CountUp** - حركات أرقام متحركة
- **next-intl** - دعم تعدد اللغات (i18n)

### أدوات التطوير
- **pnpm** - مدير حزم سريع وفعال
- **Tailwind CSS** - معالج CSS متقدم
- **PostCSS** - معالج CSS
- **Autoprefixer** - إضافة بادئات المتصفحات تلقائياً

## 🚀 البدء السريع

### المتطلبات
- Node.js 18+ أو أحدث
- pnpm أو npm

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/YOMNA190/al-habeeb-website.git
cd al-habeeb-website

# تثبيت المكتبات
pnpm install
# أو
npm install
```

### التشغيل في بيئة التطوير

```bash
pnpm dev
# أو
npm run dev
```

ثم افتح المتصفح على: `http://localhost:3000`

### البناء للإنتاج

```bash
pnpm build
pnpm start
# أو
npm run build
npm start
```

## 🎨 نظام التصميم

### الألوان الأساسية

| الاسم | الكود | الاستخدام |
|-------|------|-----------|
| Navy | `#0A1628` | اللون الأساسي الداكن |
| Obsidian | `#1A1A1A` | خلفيات ثانوية |
| Gold | `#C8A45C` | اللون الأساسي للعلامة التجارية |
| Gold Light | `#E8D5A3` | تسليط الضوء والتفاعلات |
| Concrete | `#E0DDD7` | نصوص فاتحة |

### الخطوط

- **العربية:** Tajawal (للعناوين والنصوص الرئيسية)
- **الإنجليزية:** Inter (للنصوص الثانوية)

### الحركات

- **مدة الحركات:** 200-500ms حسب نوع التفاعل
- **دالة التخفيف:** cubic-bezier(0.23, 1, 0.32, 1) للحركات السلسة
- **الحركات المدعومة:** Fade, Slide, Scale, Rotate
